# 🌸 FocoQuiz

Quiz educativo de matemática desenvolvido especialmente para pessoas com TDAH e deficiência intelectual leve.

## 💡 Sobre o projeto

Criado para minha irmã de 16 anos, que tem TDAH e deficiência intelectual leve, o FocoQuiz foi pensado para tornar o aprendizado de matemática mais leve, divertido e acessível.

Pessoas com TDAH sentem tédio rápido e se dispersam com facilidade. Por isso o projeto foi desenvolvido com:

- Sessões curtas de 5 perguntas
- Feedback imediato e positivo
- Visual colorido e acolhedor
- Sem punição por erro — só aprendizado

## 🎯 Funcionalidades

- Tela de boas vindas personalizada com o nome do jogador
- Quiz de operações básicas e tabuada
- Barra de progresso visual
- Feedback de acerto e erro com cores
- Pontuação em tempo real
- Tela de resultado com pontuação final
- Botão para jogar novamente
